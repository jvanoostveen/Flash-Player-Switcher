How does this work?
When plugins are placed in the users 'Internet Plug-Ins' folder, those are loaded instead of the system 'Internet Plug-Ins'. As this tool places files inside the users library, no password is needed.
Note: It is best to disable the automatic updating of the Flash Player plugin.
I'm not seeing any changes...
You need to restart your browser to load the new plugin. If you're using Chrome, disable the built-in Flash Player plugin. Please note that installing a Flash Player plugin by using the installer will remove the plugin placed in the user library directory.
How do I add plugins?
There is a 'Download more…' button where it is possible to download some prepared plugins. If you need different versions or you don't trust the source (nothing was changed, but if you want to be really sure), you can download plugins from the Adobe website , install them and use the tool to add those plugins to the storage. Make sure you use the uninstaller when trying to install older versions of the plugin when you get errors during the install.
Can I change the location of the plugin storage?
In the preferences window the storage location can be changed to whatever directory you choose. All current data and plugins is moved to this new location.
Can I share a plugin storage?
Although yet untested, you could change the location of the storage to a synced directory (for example using Dropbox ). One person should be the maintainer of the storage location and others us the location as read-only. Uncheck the allow editing checkbox to disable changing the storage data and plugins.
